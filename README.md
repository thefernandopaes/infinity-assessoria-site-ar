# Infinity Assessoria

Site institucional da Infinity Assessoria — tecnologia, automação de fluxos e integração de sistemas para empresas.

## Stack

- React 18 + TypeScript
- Vite (SWC)
- Tailwind CSS 3 + shadcn/ui
- React Router DOM

## Desenvolvimento

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
